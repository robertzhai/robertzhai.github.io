## php-yii2-hui-cms-boilplate
>基于yii2和hui的一个cms开发样例boilplate，yii2开发基于gii还是很快的，感觉比codeigniter开发
要方便。clone本项目即可基于已经完成的样例代码进行快速开发、部署。

**[**基于**]**


* php 5.6
* nginx
* mysql
* yii2
* hui css
* jquery
